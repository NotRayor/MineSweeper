## 구현해야할 기능
1. 지뢰 폭팔 애니메이션 구현
2. 버튼을 꾹 누르면, 깃발 설치
3. 화면 상단부에, 타이머와 남은 지뢰 갯수 표시
4. 처음 클릭 시, 지뢰, 인접타일 선택되는 경우가 없도록 구현

## 문제점
